# Mode NeuroClip

| Mode | Fungsi |
|---|---|
| default | jawaban singkat umum |
| form | jawaban siap tempel ke Google Form |
| pilihanganda | jawaban + display + alasan |
| opsi | hanya huruf opsi |
| sd | gaya anak SD |
| smp | gaya siswa SMP |
| sma | gaya siswa SMA |
| singkat | 1-2 kalimat |
| sedang | 2-4 kalimat |
| lengkap | jawaban panjang dan runtut |
| formal | bahasa formal |
| code | solusi coding |
| math | langkah hitung |
| wa | format WhatsApp |
| ringkas | ringkas teks |
| rewrite | tulis ulang |

## Set mode

```bash
neuro mode form
neuro mode sd
neuro mode code
```

## Lewat menu

Tap notif lalu isi:

```txt
mode form
mode sd
mode pilihan ganda
```
